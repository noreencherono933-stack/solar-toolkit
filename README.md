Project Overview
The SolarPulse Toolkit is a specialized interface for solar energy systems. It provides engineers and users with a high-level overview of system health, power generation trends, and efficiency metrics using a responsive React-based UI.

Tech Stack
* Frontend:React.js (Vite)
* Styling: Tailwind CSS (Utility-first)
* Visualization: Recharts (SVG Charts)
* Icons: Lucide-React
* Deployment: Netlify / Vercel

 Key Features
* Real-time Metrics: Displays Current Output (kW), Battery Storage %, and Panel Efficiency.
* Interactive Analytics: A dynamic Area Chart tracking the diurnal solar yield cycle.
* Responsive Design: Optimized for both mobile field inspections and desktop monitoring stations.
* System Diagnostics: Visual heartbeat indicator showing system "Active" status.
